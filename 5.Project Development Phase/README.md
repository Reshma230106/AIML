# Project Development Phase

## Overview
The SmartLender application was developed using Python, Flask, and Machine Learning.

## Development Process
1. Dataset collection and analysis.
2. Data preprocessing.
3. Training machine learning models.
4. Evaluating model performance.
5. Selecting the best-performing model.
6. Integrating the trained model with Flask.
7. Developing the web interface.

## Machine Learning Models
- Decision Tree
- Random Forest
- K-Nearest Neighbors (KNN)
- XGBoost

## Best Model
XGBoost achieved a test accuracy of 84.0%.

## Technologies Used
- Python
- Flask
- HTML
- CSS
- Machine Learning

## Running the Application
Install the required packages and run:

python app.py
