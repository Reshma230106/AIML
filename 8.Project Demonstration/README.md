# Project Demonstration

## Project Title
SmartLender – Loan Applicant Creditworthiness Prediction

## Demonstration Overview
The SmartLender project demonstrates how machine learning can be used to predict the creditworthiness of loan applicants.

## Application Workflow
1. User opens the SmartLender web application.
2. User enters the required loan applicant details.
3. The application processes the input data.
4. The trained machine learning model makes a prediction.
5. The prediction result is displayed to the user.

## Machine Learning Models
- Decision Tree
- Random Forest
- K-Nearest Neighbors (KNN)
- XGBoost

## Best Model
XGBoost achieved a test accuracy of 84.0%.

## Project Outcome
The SmartLender application successfully demonstrates an AI/ML-based approach for predicting loan applicant creditworthiness.
