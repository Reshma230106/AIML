# Project Design Phase

## System Architecture
The SmartLender system uses a machine learning model integrated with a Flask web application to predict loan applicant creditworthiness.

## System Flow
1. User enters applicant details.
2. The web application receives the input.
3. Input data is processed.
4. The trained machine learning model makes a prediction.
5. The prediction result is displayed to the user.

## Main Components
- User Interface
- Flask Backend
- Data Preprocessing
- Machine Learning Model
- Prediction Result

## Machine Learning Models
- Decision Tree
- Random Forest
- K-Nearest Neighbors (KNN)
- XGBoost

## Best Model
XGBoost achieved a test accuracy of 84.0%.
