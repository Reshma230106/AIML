# Project Documentation

## Project Title
SmartLender – Loan Applicant Creditworthiness Prediction

## Project Description
SmartLender is a machine learning-powered web application developed to predict the creditworthiness of loan applicants.

## Technologies Used
- Python
- Flask
- HTML
- CSS
- Machine Learning

## Machine Learning Algorithms
- Decision Tree
- Random Forest
- K-Nearest Neighbors (KNN)
- XGBoost

## Best Model
XGBoost achieved a test accuracy of 84.0%.

## Documentation
This folder contains the complete project documentation, reports, presentation files, and other supporting documents related to the SmartLender project.

## Conclusion
SmartLender provides an efficient machine learning-based approach for predicting loan applicant creditworthiness.
