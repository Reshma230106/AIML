# Project Testing

## Testing Overview
The SmartLender application was tested to ensure that the machine learning model and web application work correctly.

## Testing Activities
- Tested user input fields.
- Tested data preprocessing.
- Tested machine learning predictions.
- Tested Flask application functionality.
- Verified prediction results.
- Checked application performance.

## Model Testing
The following machine learning models were evaluated:

- Decision Tree
- Random Forest
- K-Nearest Neighbors (KNN)
- XGBoost

## Test Result
XGBoost achieved the best test accuracy of 84.0%.

## Conclusion
The SmartLender application successfully predicts the creditworthiness of loan applicants.
