# Requirement Analysis

## Functional Requirements
- User can enter loan applicant details.
- System predicts applicant creditworthiness.
- Machine learning models process the input data.
- System displays the prediction result.

## Non-Functional Requirements
- Easy-to-use interface.
- Fast prediction results.
- Reliable and accurate performance.
- Secure handling of user data.

## Software Requirements
- Python
- Flask
- Machine Learning Libraries
- HTML
- CSS

## Hardware Requirements
- Computer/Laptop
- Minimum 4 GB RAM
- Internet connection

## Dataset
The dataset contains applicant information used to train and test the machine learning models.
